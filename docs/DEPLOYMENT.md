# Deployment Guide

This guide covers deploying the Basketball Portfolio to various platforms.

## ğŸš€ Quick Deploy Options

### **Netlify (Recommended)**
1. **Connect GitHub Repository**
   - Go to [Netlify](https://app.netlify.com/)
   - Click "New site from Git"
   - Connect your GitHub account
   - Select the `basketball-portfolio` repository

2. **Configure Build Settings**
   ```
   Build command: yarn build
   Publish directory: out
   ```

3. **Environment Variables**
   ```
   NEXT_PUBLIC_ANALYTICS_ID=your_ga_measurement_id
   ```

### **Vercel**
1. **Install Vercel CLI**
   ```bash
   npm i -g vercel
   ```

2. **Deploy**
   ```bash
   vercel --prod
   ```

### **GitHub Pages**
1. **Update next.config.js**
   ```javascript
   module.exports = {
     output: 'export',
     trailingSlash: true,
     images: {
       unoptimized: true
     }
   }
   ```

2. **Add GitHub Pages Workflow**
   - Use the provided `.github/workflows/deploy.yml`
   - Configure secrets in GitHub repository settings

## ğŸ”§ Local Setup

1. **Clone Repository**
   ```bash
   git clone https://github.com/your-username/basketball-portfolio.git
   cd basketball-portfolio
   ```

2. **Install Dependencies**
   ```bash
   yarn install
   ```

3. **Environment Setup**
   ```bash
   cp .env.sample .env.local
   # Edit .env.local with your values
   ```

4. **Development Server**
   ```bash
   yarn dev
   ```

5. **Production Build**
   ```bash
   yarn build
   yarn start
   ```
